
# 🚔 Interventie Portaal - Polaris
 
Een responsive webapplicatie voor een fictieve politiedienst, ontwikkeld als eindproject voor school. Het project is geïnspireerd door systemen zoals het [Nederlandse MEOS](https://nl.wikipedia.org/wiki/Politie_in_Nederland#Gebruikte_systemen) systeem, die politieagenten ondersteunen om **efficiënter en mobieler** te werken.

## Over het project

Polaris is een webapplicatie waarmee meldingen van incidenten kunnen worden geregistreerd en beheerd. Het project simuleert een gecentraliseert, intern systeem dat gebruikt zou kunnen worden door politieagenten om interventies op te volgen. Ik heb gekozen voor dit onderwerp om aan te tonen om te tonen hoe digitale tools kunnen bijdragen aan efficiëntere werkprocessen binnen hulpdiensten. En om aan te tonen dat, [I-Police](https://www.computable.be/2025/12/28/i-police-mislukt-politie-keert-terug-naar-gefaseerde-modernisering/), toch nog mogelijk is al dan niet op een andere manier.

De applicatie bestaat uit:

- Een publiek gedeelte voor meldingen
- Een beveiligd intern portaal voor agenten/medewerkers

## 🎯 Doelstellingen
- Een toegankelijke, makkelijk te leren web-app.
- Gegevens correct en beveiligd opslaan/ophalen uit een databank.
- Werken met een gestructureerde backend. 
- Realistische en professionele applicatie bouwen.
- Inzicht tonen in hoe digitale systemen operationele processen ondersteunen
